Campos de input

Text: um campo de texto curto, normalmente contando com os atributos maxlength e minlength, para definir a máxima e a mínima quantidade de caracteres do texto.

Button: um simples botão, meio triste até

Checkbox: Uma checkbox, podendo ser marcada previamente usando o atributo checked. É necessário usar a tag div para criar as boxes

Escolha uma cor:

Date: Input simples de data, com dia, mês e ano. Pode ter os atributos de min, max e value, por exemplo.

Datetime(local): Input de data e horário(hora, minuto, segundo) sem fuso horário

Email: Campo de texto para um endereço email

File: Input para permitir o usuário anexar um arquivo. Se utiliza o atributo accept para definir os tipos de arquivo a serem recebidos

Number: Controle de números. Só isso. Pode ter atributos de max e min

Password: Campo de texto em sigilo, para inserir senhas. Com o atributo maxlength e minlength é possível especificar o tamanho máximo e mínimo do valor inserido

Radio: Botão para escolher uma opção. Necessita o atributo value para definir o valor do item. O atributo checked pode ser usado também.

Escolha um:

Reset: Faz os atributos e valores do forms serem resetados

Range: Sla algum controle acho? Não precisa de valores específicos
